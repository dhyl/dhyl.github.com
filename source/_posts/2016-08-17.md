# 安装
------
> * `git clone https://github.com/dhyl/dhyl.github.io.git themes/yelee`
> * 修改 Hexo 根目录对应配置文件 _config.yml `theme: Yelee`
 ![](/yelee.png)
> * 执行以下命令预览当前主题 `hexo clean && hexo s`
# 更新
------
> * 执行以下两条命令
> *                               `cd themes/yelee`
> *                               `git pull`
